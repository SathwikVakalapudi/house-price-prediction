# House Price Prediction

A Machine Learning project to predict house prices using Python and common ML libraries such as Pandas, Scikit-learn, Matplotlib, and Seaborn.

This repository contains a Jupyter Notebook (`Untitled.ipynb`) that loads a housing dataset, performs Exploratory Data Analysis (EDA), feature preprocessing, model training, evaluation, and prediction.
